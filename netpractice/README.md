# NetPractice Project Summary

## Project Overview
NetPractice is a practical networking exercise designed to teach fundamental TCP/IP network configuration skills. The project presents a web-based interface where students must solve networking problems by configuring small-scale networks to establish proper connectivity between devices.

## Key Features & Requirements

### Core Concept
- **Network configuration exercises** through a web-based training interface
- **10 progressive levels** of increasing complexity
- **Virtual network simulation** without requiring physical hardware
- **TCP/IP addressing fundamentals** as the core learning objective

### Technical Components
- **Web interface** (index.html) accessible via web browser
- **Network diagram visualization** showing non-functioning networks
- **Configuration fields** that require modification to establish connectivity
- **Real-time validation** through "Check again" functionality
- **Configuration export** via "Get my config" button for submission

### Learning Objectives
- Understanding **TCP/IP addressing** schemes
- Configuring **subnet masks** and **IP addresses**
- Setting up **routing** between network segments
- Managing **network device configurations**
- Troubleshooting **network connectivity issues**

### Exercise Structure
Each level presents:
- **Clear objectives** displayed at the top of the window
- **Network diagrams** with configurable devices
- **Interactive configuration fields** for IP addresses, subnet masks, and routes
- **Diagnostic logs** providing feedback on configuration errors
- **Progressive difficulty** from basic to complex network scenarios

## Submission Requirements
- **10 configuration files** (one per completed level)
- **Files placed at repository root**
- **Intranet login** must be used during practice sessions
- **Configuration export** using the "Get my config" button for each level

## Evaluation Process
- **Defense component**: 3 random levels from 6-10 must be solved within 15 minutes
- **No external tools** allowed during evaluation (basic calculator tolerated)
- **Practical demonstration** of networking knowledge under time constraints
- **Configuration accuracy** tested through the web interface

## Technical Skills Developed
- **IP address allocation** and subnet calculation
- **Router configuration** and routing table management
- **Network segmentation** understanding
- **Troubleshooting methodology** for network issues
- **TCP/IP protocol stack** practical application

This project emphasizes hands-on learning of networking fundamentals through practical exercises, preparing students for real-world network administration tasks by developing critical thinking and problem-solving skills in TCP/IP network configuration.
