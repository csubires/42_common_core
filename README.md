# 42 CURSUS
	42 Campus is an innovative educational project in programming and software development that has gained popularity for its non-traditional approach. This initiative was originally created in France by entrepreneur Xavier Niel and has expanded to several countries, including Spain.
- There are no teachers or traditional classes. Students learn through practical and collaborative projects.
- It uses a peer-to-peer learning system, where students learn from each other and evaluate each other.
- The curriculum is based on progressive projects that allow students to develop skills in programming, algorithms, and logical thinking.
